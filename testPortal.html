The Students Portal

The students portal

Here the student can see old test-results or start a test if the scheduled time has arrived.

Running a test

When performing a test the student have the option to answer the questions in any order they like. Questions which have already been answered are marked in grey. All given answers are saved in the database, so if the browser i closed by mistake, the student can always restart the test and continue from where they left off.

Submit

A test can only be submitted after all questions have been answered. The student can also attach a comment for the teachers. When a test has been submitted it can no longer be restared by the student.

Result

Immediately upon submission, the student is given the result of the test.

Updated Student Portal

The students portal is immediately updated with the new testscore.

The Teachers Portal - Managing Tests

Schedule test

Tests can be scheduled and groups/individual students that should have accesss to the test can be selected.

Test status

When selecting "status" for a test on the schedule test page, it is possible to see which students has started and finished the test

Archive tests

When a test is archived any scheduled test-times are removed. There are no hard delete implemented, so an archived test can always be reactivated.

Student result

Review all the tests a student has performed. Clicking the Test-button will show all the questions from the selected test with correct and given answer.
