I studied at Lexicon from march 2024 until january of 2025. The curricula was mostly focused on C# and .NET / ASP.NET, but we also got some insights into frontend, databases and Azure.

CyberdyneMoviesInc

This was a short group project. The asignment was to create a movie webshop. If you don't want to create a user (by giving an email) you can use gandalf@example.com to try out the customer part of the website.

C#, ASP.NET MVC, EntityFramework, Bootstrap.

OmegaNews

A lager group project where we created a news website. Use login "gandalf@example.com" and password "Password123" to login as a user with a subscription.

C#, ASP.NET MVC, Identity, EntityFramework, Bootstrap, JavaScript, Azure

Generate Excercises

A small project I did when we were learning about Azure AI. This app uses a GTP model to generate different types of excercises with solutions. The responses are formatted in HTML by the AI.

C#, ASP.NET, Azure AI

The Test Portal

My final project at Lexicon

The school was still giving exams on paper, and to fix this issue we created a webb-app for both teachers and students. Teachers can create and schedule tests, while students can log in, take tests and immediately see their results. Everything is saved in a database.

We were two people working on this project; my colleague worked on the identity/authorization part of the project, while I did the rest.

The application are currently used in production at Lexicon.

Fullstack ASP.NET with entity framework
